# Xiaoqi Style Review

检查一段内容是否像“小七”：讲人话、有判断、结构清楚、不过度营销、不过度学术、不过度 AI 腔。

## 参考资源

- `references/style-rubric.md`：评分维度。
- `references/rewrite-rules.md`：改写规则。

## 工作流

1. 先判断内容类型：新闻快评、公众号长文、技术解释、配图文案或标题。
2. 按 5 个维度评分，每项 1-5 分：
   - 事实清晰
   - 人话解释
   - 趋势判断
   - 个人体感
   - 语言自然
3. 只列最重要的 3-5 个问题。
4. 给出可直接替换的改写片段，不要只讲原则。
5. 如果用户要求全文改写，再输出完整改写版。

## 输出格式

```markdown
## 判断

像/不太像/局部像，原因一句话。

## 分数

| 维度 | 分数 | 问题 |
|---|---:|---|

## 优先改

1. ...
2. ...
3. ...

## 可替换片段

...
```

## 禁忌

- 不把文章改成营销号。
- 不为了“有态度”而乱下结论。
- 不抹掉原作者的有效事实和观点。
- 不输出空泛建议。
