# 审校与改写规则

## 保留顺序

1. 保留已被材料支持的事实、数字和引用。
2. 保留作者真实立场、语气强度和有效个人经验。
3. 先修正事实与判断混淆，再处理结构、句子和词语。
4. 优先删除无信息内容；不要用另一句套话替换套话。

## 去 AI 味负面清单

以下特征单独出现不一定有问题；重复、成组出现并挤压信息时才判为明显 AI 腔。

### 模板路标

- 反复使用“值得注意的是”“真正关键的是”“这意味着什么”“不难发现”。
- 每节都按“先说结论—拆三点—再总结”运行。
- 用“首先、其次、最后”制造并不存在的逻辑层级。

### 虚假笃定

- 使用“毫无疑问、必将、不可逆、注定”等词，却没有证据或条件。
- 把产品演示、官方宣称、相关性或短期数据写成已证实因果。
- 用“业内普遍认为”“大家都知道”隐藏来源缺失。

### 空洞升华

- 使用“不仅是……更是……”“这背后折射出……”把普通更新抬成时代转折。
- 结尾突然落到“未来已来、拥抱变化、让我们拭目以待”。
- 重复前文观点，却没有新增边界、行动或观察信号。

### 过度规整

- 连续三段拥有相同长度、相同句式或相同收尾。
- 为凑“三点”拆散一个观点，或为对称标题扭曲内容。
- 形容词、四字词和口号并列，具体动作与结果过少。

### 伪个人化

- 添加原稿没有的“我亲测、我的体感、我一直在关注”等经历。
- 用密集“我觉得”假装有态度，但判断没有依据。
- 强行设置互动提问、关注转发或煽情结尾。

## 改写动作

- 长句拆成能独立承载一个因果的句子。
- 把抽象词换成主体、动作、对象和结果。
- 将来源方口径改为有归属的表达。
- 给判断补依据、条件或反例；无法补时降低确定性或删除。
- 调整句长和段落节奏，但不追求刻意口语化。
- 改完后与原稿对照，确认没有新增事实、体验和立场。
