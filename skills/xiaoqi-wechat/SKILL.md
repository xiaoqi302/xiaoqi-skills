# Xiaoqi WeChat

把可靠素材写成清楚、有判断、像真人交流的中文公众号长文。核心是“事实 + 人话解释 + 有边界的判断”，不是套用固定句式。

## 参考资源

写作前读取 [公众号写作规则](references/writing-guide.md)，按其中的结构、事实边界和反模板规则执行。

## 工作流

1. 明确读者、产物和核心问题；保留用户已经给出的立场。
2. 将素材分为事实、来源方主张、作者判断和待确认信息。
3. 判断事实覆盖度：
   - 核心事实充分：重组并完成全文。
   - 核心事件明确但细节不足：只补结构、解释和段落衔接，显式保留缺口。
   - 连事件主体都不清楚：只问一个会改变成稿方向的必要问题。
4. 围绕一个中心判断组织文章；每个二级标题只推进一个问题。
5. 写完后检查来源、日期、数字、引语和因果关系，删除无法由素材支撑的内容。
6. 仅当用户明确需要封面或正文配图时，再分别使用 $xiaoqi-cover-image 或 $xiaoqi-article-illustrator。

## 输出

默认直接交付完整文章，不解释改写过程，也不自动推销配图。若事实不足，只交付可由现有材料支撑的正文或结构，并列出最少量的待补事实。

## 底线

- 不补写素材中不存在的数据、来源、引语、案例、个人经历或确定性结论。
- 不把来源方宣传口径写成已验证事实。
- 不用年份、固定转折、固定结尾或成组金句制造“小七感”。
- 不为了增强态度而改变用户原意。
