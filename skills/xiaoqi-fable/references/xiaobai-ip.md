# 小白 IP

## 角色定义

小白是小七寓言配图的固定视觉 IP。

默认每张图都要出现小白。小白不是吉祥物，不是贴纸，不是讲师，而是正在参与结构变化的冷静行动者。

## 外形

- 白色或近白色空脸，脸部简单干净。
- 眼睛可以是两个很小的黑点，也可以没有明显表情。
- 深色短斗篷、长外套或简单工作服，可以有衣领。
- 细手脚，鞋子极简。
- 轮廓略微不规则，有手绘感。
- 形体不能像已有影视、动漫、游戏角色。

服装可以变化，但只服务动作和场景：观察员、搬运者、修补者、记录员、守门人、换座者、测量者。不要复杂 cosplay。

## 性格

- 冷静、克制、观察后行动。
- 有一点未来感，但不是机器人。
- 不卖萌，不夸张，不做表情包。
- 像一个在寓言系统里认真修正现实的人。

## 常见职责

让小白承担核心动作：

- 观察远处窗口。
- 给系统刻度重新校准。
- 在门槛前递出钥匙。
- 把椅子从旧桌移到新桌。
- 给断裂的桥补一块木板。
- 在风险天平上放下砝码。
- 用小旗标记转折点。
- 在雾里展开地图。
- 把错误的箭头擦掉重画。
- 守住一扇还没有完全打开的门。
- 从旧书堆走向田野采样。
- 把过时规则折成纸船放走。

## 禁止

- 不要画成已有 IP 或影视角色。
- 不要画成机器人、魔法师、侦探、骑士或复杂职业装。
- 不要过度可爱、圆润、商业化。
- 不要夸张表情、大眼睛、闪亮眼。
- 不要让小白只是站在角落里看。
- 不要让小白抢走文章结构表达。

## 判断标准

如果去掉小白，图的核心隐喻还能完全成立，说明小白太装饰了；要重写提示词，让小白成为动作主体。

## 系列一致性

- 有用户参考图：先查看，再作为每张图的共同参考；不要凭文件名猜造型。
- 无参考图：固定空脸形状、深色外套轮廓、细手脚比例和黑白红蓝配色，写成一段角色锚点并逐字复用。
- 工具支持时固定种子，或把首张通过质检的图用于后续参考；不支持时不要声称已锁定参考图。
- 场景和动作可以变化，脸、身体比例、外套主轮廓与克制气质不能漂移。
