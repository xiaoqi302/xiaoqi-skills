# 风格 DNA

## 一句话

纯白、黑白手绘、留白、克制、未来寓言、轻微木刻感、少量红蓝批注、结构清楚但不说明书。

像一个长期观察 AI、技术周期、商业变化和个人选择的人，在白纸上画下的一则现代寓言。

## 必须

- 16:9 横版正文配图，除非用户指定其他比例。
- 纯白或极浅近白背景：干净、安静、没有复杂纹理。
- 黑色手绘线稿为主：细线、轻微抖动、局部排线或木刻质感，不机械、不矢量。
- 大量留白：主体占画面约 35%-55%，至少 40% 空白。
- 少量中文手写批注：每张 3-6 处，每处尽量 2-8 个字。
- 一张图只讲一个核心判断、结构、状态或隐喻。
- 画面要像寓言现场，不像讲义页面。

## 颜色

- 黑色：主体线稿、角色、物件、框线、主要文字。
- 红色：风险、转折、警告、错误、必须补救的地方。
- 蓝色：系统状态、第二层说明、概率、冷静旁注。
- 橙色：只在需要表达路径或移动关系时少量使用。

颜色要非常克制。宁可只有黑色和一两处红蓝，不要把画面变成彩色信息图。

## 绝对不要

- 不要商业插画。
- 不要 PPT 信息图。
- 不要正式流程图。
- 不要科技感 UI 或科幻海报。
- 不要国风装饰海报。
- 不要儿童绘本或可爱卡通。
- 不要复杂架构图。
- 不要精致扁平插画。
- 不要真实 App 截图。
- 不要深色背景、渐变、强阴影、重噪点。
- 不要把每个节点都解释清楚。
- 不要左上角写“趋势分析 / 系统架构图 / 观点图解 / 流程图”等类型标题。

## 审美方向

要冷静、清醒、微妙、有寓言意味，像在讲一个未来时代的日常小场景。

不要卖萌、不要热血、不要宏大叙事、不要太满。
