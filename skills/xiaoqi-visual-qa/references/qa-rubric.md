# 视觉质检量表

## 结论与严重度

### 结论

- `PASS`：无 P0、无阻塞交付的 P1；可以保留不影响使用的轻微 P2。
- `REPAIR`：主体、构图和风格基本成立，问题有明确局部区域且可稳定编辑。
- `REGENERATE`：存在多处结构性错误、大面积乱码、错误技术关系、严重裁剪或整体风格/IP 失效，局部修复代价高或结果不可靠。

### 严重度

- `P0`：会造成事实错误、品牌风险、错误操作或无法使用。包括必须准确文字错误、关键内容被裁、技术箭头/关系错误、冒充真实 Logo、明显水印等。
- `P1`：明显破坏理解或目标风格。包括主次颠倒、关键文字难读、画面过满、核心角色漂移、比例不适配平台等。
- `P2`：不阻塞交付的局部瑕疵。包括轻微对齐、间距、线条或色彩波动。

## 检查维度

| 维度 | 必查内容 | 图中证据写法 | 常见判定 |
|---|---|---|---|
| 比例与裁剪 | 实际像素比例、平台安全区、标题/人物/边缘元素是否被切 | 写明尺寸与被裁区域，如“右侧人物手臂贴边并被截断” | 关键内容被裁为 P0；比例不适配但可补画布为 P1 |
| 文字准确与可读性 | 必须准确文字逐字比对；错字、乱码、漏字、大小、对比度 | 抄录实际可见错误，并标明位置；看不清就写“无法辨认”，不要猜 | 关键文字错为 P0；次要标签模糊通常为 P1 |
| 视觉层级 | 标题、核心主体、辅助信息的阅读顺序是否清楚 | 指出最先吸引视线的元素及其与目标的冲突 | 核心信息被装饰抢走通常为 P1 |
| 留白与密度 | 元素数量、边距、拥挤区、无效空洞、手机缩略图可读性 | 指出拥挤或失衡区域，不用抽象形容词 | 信息互相遮挡为 P1；轻微间距为 P2 |
| 风格一致 | 背景、线条、色彩、材质、图标和字体是否符合目标风格 | 对照已给规格或已查看参考图，列出具体偏差 | 整体风格错误可判 REGENERATE；局部漂移为 P1/P2 |
| 技术关系 | 节点、方向、顺序、因果、数据流、公式和数值是否与目标一致 | 指出具体节点与箭头，如“A 到 B 的箭头画反” | 错误关系或数字为 P0；非技术图标为 N/A |
| IP 角色一致 | 脸型、比例、服装轮廓、配色、标志动作是否与已查看参考一致 | 写出当前角色与参考图的可见差异 | 角色已像另一个 IP 或系列断裂为 P0/P1 |
| 伪 Logo 与水印 | 生成器水印、签名、二维码、仿品牌标识、随机字母角标 | 标明位置、形状和可见字符；不要猜品牌来源 | 冒充品牌或明显水印为 P0；无意义角标通常为 P1 |

## 证据规则

- 使用“左上、中央、右下、标题下方、人物右侧”等可复现位置。
- 只描述实际看到的像素，不把提示词目标写成成图事实。
- 文字无法可靠辨认时，记录为不可读，不自行补全。
- 未提供平台、风格或参考图时，标记未验证；不能因此宣称对应维度通过。
- 参考图必须实际查看后才可用于风格或 IP 一致性结论。
