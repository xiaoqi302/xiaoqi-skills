# 主题笔记图质检

## 内容

- 标题仍是用户给出的主题，不被改成文学化口号。
- 辅助标签来自用户材料、稳定常识或已核验来源，没有为凑模块虚构事实。
- 一张图只围绕一个主题，模块不超过 5 个，不扩成教程或知识大全。

## 画面

- 比例为竖版 3:4，或严格采用用户指定比例。
- 标题在手机缩略图中可读；中文无错字、乱码和模糊笔画。
- 文字只用短词或短语，插图承担主要解释，留白明显。
- 不出现多页拼贴、密集思维导图、PPT 风或正式海报风。

## 修复顺序

1. 修正标题、错字和未经核验的事实标签。
2. 删除长句、重复标签和无关装饰。
3. 放大标题与插图，恢复留白。
4. 局部编辑无效时再重生成。

每张图最多修复两轮；仍未通过时交付最优版本并说明缺陷。
