# Xiaoqi Topic Note

把一个主题直接转成竖版中文手绘笔记图。用户输入的内容就是主题，不改写、不扩写、不追问。

## 工作流

1. 读取用户原始输入作为 `{主题}`。
2. 默认生成 1 张图；只有用户明确要求时才生成多张。
3. 默认比例为 `3:4`，竖版，中文，大字体，低文字密度。
4. 调用当前环境可用的图像生成工具；如果没有工具，只输出可直接使用的完整提示词。
5. 生成后检查：是否围绕主题、中文可读、文字少、留白足、模块不超过 5 个。

## 固定提示词

将 `{主题}` 替换为用户输入的原始主题：

```text
请生成一张小七风格中文手绘主题笔记图，主题是：【{主题}】。

画面要求：
- 竖版 3:4，4K，适合手机阅读。
- 大标题必须明显偏大，手机屏幕上一眼能读清。
- 总文字量很少，只保留关键词、短标签和极短注释。
- 最多 3-5 个重点模块，每个模块只放 1-3 个短词或短语。
- 留白充足，版面疏朗，不能塞满。
- 使用黑色手写笔记线条，少量红色/蓝色/黄色马克笔高亮。
- 插图比文字更重要，用手绘小图解释主题。
- 可以有箭头、波浪线、圈注、便签和剪报感元素，但注释必须稀疏。
- 中文字体要清晰，禁止密集小字、长段落、PPT 风和正式海报风。

输出：一张完整图片，不要拼贴多页，不要英文主文案。
```

## 输出

- 如果已生成图片，返回图片路径和主题。
- 如果只能输出提示词，直接给完整提示词，不额外解释。

## 禁忌

- 不把主题改成更文学化的标题。
- 不扩成教程、长文、思维导图或 PPT。
- 不生成密密麻麻的小字。
- 不把中文提示词翻译成英文，除非图像工具强制需要。
