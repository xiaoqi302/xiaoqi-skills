# Prompt Patterns

Use these templates as starting points. Replace bracketed fields.

## Universal Header

```text
Create one single original logo mark on a pure white background.

Product context: [one sentence product description].
Audience: [target users].
Tone: [3-5 words].
```

## Modular Token

```text
Direction: Modular Token.
Motif: [specific mechanism].
Shape grammar: an irregular cluster of [3-6] simple tokens forming one strong silhouette, with one missing or offset token as the visual hook. Avoid standard 2x2 or 3x3 grid, app launcher, window grid, Figma-like vertical blocks, and generic component icon.
Color: [bright but mature palette], flat colors only.
Style: vector-like, crisp edges, minimal, recognizable at 32px, no text, no labels, no mockup, no shadows, no 3D, not copying any existing brand.
```

## Cutout Mass

```text
Direction: Cutout Mass.
Motif: [specific mechanism].
Shape grammar: one monolithic silhouette with one memorable carved cut or slot. The cut should not read as play button, lightning bolt, arrow, folder tab, or letter. Use one hard edge and one soft corner to make the mark ownable.
Color: [single color or two cohesive colors], no decorative gradient.
Style: vector-like, crisp edges, minimal, recognizable at 32px, no text, no labels, no mockup, no shadows, no 3D, not copying any existing brand.
```

## Folded Band

```text
Direction: Folded Band.
Motif: [specific mechanism].
Shape grammar: one continuous folded band with consistent angle logic and one asymmetric negative-space channel. It must feel like one object, not several panels. Avoid portal made from separate slabs, ribbon decoration, arrow, or readable letter.
Color: [cohesive gradient or flat 2-color system].
Style: vector-like, crisp edges, minimal, recognizable at 32px, no text, no labels, no mockup, no shadows, no 3D, not copying any existing brand.
```

## Linear Emblem

```text
Direction: Linear Emblem.
Motif: [specific mechanism].
Shape grammar: one consistent-stroke emblem with one offset joint or gap as the hook. It should be app-icon strong, not a UI icon, folder, code bracket, or chart.
Color: [single bright or dark color] with optional tiny integrated accent.
Style: vector-like, crisp edges, minimal, recognizable at 32px, no text, no labels, no mockup, no shadows, no 3D, not copying any existing brand.
```

## Geometric Aperture

```text
Direction: Geometric Aperture.
Motif: [specific mechanism].
Shape grammar: [4-6] hard-edge or mixed-edge forms around an asymmetric negative space. Avoid flower, star, camera shutter, swirl, central round hole, or wave.
Color: [warm or bright color family], flat or very restrained gradient.
Style: vector-like, crisp edges, minimal, recognizable at 32px, no text, no labels, no mockup, no shadows, no 3D, not copying any existing brand.
```

## iOS App Icon Adaptation

```text
Create an iOS app icon version of the selected logo mark.
Keep the exact core silhouette. Place it inside a rounded-square app icon field with a clean background.
No text, no mockup, no device frame, no extra symbols.
The mark must remain recognizable at small sizes.
```
