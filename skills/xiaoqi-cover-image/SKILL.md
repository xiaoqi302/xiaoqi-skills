# Xiaoqi Cover Image

为文章做封面，不做花哨海报。封面要在 2 秒内传达主题、冲突和气质。

## 参考资源

- `references/cover-style.md`：封面风格。
- `references/prompt-template.md`：提示词模板。

## 工作流

1. 提炼文章主题、核心判断、读者情绪和一个主视觉隐喻。
2. 默认给 3 个封面方向：
   - `概念隐喻型`
   - `人物观察型`
   - `结构对比型`
3. 默认比例：
   - 公众号封面：`2.35:1` 或用户指定比例。
   - 通用文章头图：`16:9`。
   - 播客/头像封面：`1:1`。
4. 每个方向输出：
   - 标题建议
   - 画面构图
   - 文字策略
   - 图像生成提示词
5. 如果用户只要一张，选最稳的方向直接生成/输出提示词。

## 画面原则

- 封面第一眼是视觉，不是标题。
- 中文标题最多 8-14 字。
- 留白要足，主体要大。
- 小七风格优先：黑白手绘、少量红蓝批注、清醒、克制、有判断。
- 不要科技蓝渐变背景，不要霓虹城市，不要抽象光效堆叠。

## 输出格式

```markdown
## 方向 A

标题：
构图：
文字：
提示词：

## 方向 B
...

## 推荐

...
```
