# 封面平台与质检规则

## 核心风格

- 背景干净，主体清楚。
- 视觉隐喻比装饰重要。
- 可以用“小白”式观察者，也可以只用物件和结构。
- 文字少，标题像手写批注或黑体贴纸。
- 色彩克制：黑白为主，红蓝黄只做重点。

## 安全区与裁剪

- 四边至少保留约 8% 空白，不把标题、脸和关键物件贴边。
- 公众号 `2.35:1` 封面把标题和核心动作放在中央主要区域，边缘只放可被裁掉的背景。
- `16:9` 视频或文章头图要同时检查中央方形裁剪，避免列表缩略图丢失主题。
- `1:1` 播客封面在小尺寸下只保留一个主视觉和一个短标题。
- 不用细线、小标签或边角信息承担核心意思。

## 视觉检查

生成后逐项检查：

1. 比例是否符合目标平台。
2. 用户提供的标题是否逐字正确，是否混入额外文字。
3. 标题、脸和主物件是否处于安全区，常见裁剪后是否仍完整。
4. 缩小到约 320px 宽时，标题和主题是否仍可辨。
5. 主体是否足够大，背景是否抢戏，是否出现水印、假 Logo 或密集小字。
6. 是否保持黑白手绘、少量重点色和克制留白。

失败时先局部编辑；标题错字、裁切关键元素或主视觉不清属于必须修复项。最多修复 2 轮。
